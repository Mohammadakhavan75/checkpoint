# AI Change Records

Every meaningful code or behavior change should create a Markdown change record and YAML metadata file.

Use:

- `ai/templates/change-record-template.md`
- `ai/templates/change-record-template.yaml`

Naming convention:

```text
AI-YYYY-MM-DD-001-short-name.md
AI-YYYY-MM-DD-001-short-name.yaml
```
